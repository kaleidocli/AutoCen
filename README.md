# AutoCen
 A small wrapper for NudeNet library.

Installation:
- python3.7 or lower
- NudeNet (https://github.com/notAI-tech/NudeNet) and its dependencies.

Stuff:
- Censoring by three options: Pixelate, Blurring, Good ol' black rectangles.
